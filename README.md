# 💼 JobConnect – Job Listing Portal

**JobConnect** is a modern frontend-only job listing portal built using **ReactJS**, designed to help users discover, search, and apply for jobs quickly and easily.

---

## 🚀 Features

- 🏠 **Home Page** – Hero section with call-to-action
- 🔎 **Job Listings** – Filter by:
  - Job title / Company
  - Location
  - Job type (e.g., Full-time, Remote)
  - Salary range (₹)
- 📄 **Job Detail Page** – Description, skills required, and "Apply Now" link
- 📝 **Application Form** – Submit name, email, and message with validation
- 💡 Clean and modern UI using **Bootstrap 5**
- 📱 Fully responsive design for desktop, tablet, and mobile

---

## 🛠️ Tech Stack

- **ReactJS** – Component-based UI
- **Bootstrap 5** – For styling and responsiveness
- **JavaScript (ES6+)**
- **JSON** – Job data stored locally (`jobs.json`)
- **Vite** – Fast dev server and build tool
